Selamat Datang di Web Saya!



Web ini berisi kumpulan tugas UAS yang dijadikan satu dalam menu utama.

Menu-menu yang terdapat dalam web ini ialah HOME, PROFIL, CSS, JAVASCRIPT, DAN PHP.

Menu HOME berisi info utama tentang web ini.
Menu PROFIL berisi PROFIL pembuat web.
Menu CSS berisi kumpulan tugas CSS diantaranya tugas 1 dan tugas 1.1.
Menu JAVASCRIPT berisi kumpulan tugas JAVASCRIPT diantaranya tugas java 1, java 2, java 3, java 4, form nilai, dan form pns.
Menu PHP berisi kumpulan tugas PHP diantaranya tugas input produk, grid produk, laporan produk, dan edit produk.
